Yep. We do track exercices.

What we can do:

  • The GET /api/users request will get all users.
  • The POST /api/users request will create a new user if you supply username with the form data/li>
  • The POST /api/users/:id/exercices with form data description, duration, and optionally date will create an exercice
  • The GET /api/users/:id/logs request will return the full exercice log of any user
  • The GET /api/users/:id/logs request accepts from, to for dates and limit for how many logs to get.
